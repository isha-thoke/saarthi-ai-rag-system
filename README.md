# saarthi-ai-rag-system


Saarthi AI RAG System


* Overview
Saarthi AI RAG System is an AI-powered Question Answering application that uses Retrieval-Augmented Generation (RAG) to provide accurate answers from PDF documents. The system extracts information from documents, stores embeddings in Pinecone, and uses Ollama LLM for generating responses.

 * Features
PDF document processing
Text chunking and embedding generation
Vector storage using Pinecone
Semantic search for relevant content
AI-powered question answering using Ollama
Fast and efficient document retrieval


* Technologies Used
Python
LangChain
Pinecone
Ollama
Sentence Transformers
PDF Processing Libraries


* How It Works
Upload a PDF document.
Extract and split text into chunks.
Generate embeddings for each chunk.
Store embeddings in Pinecone vector database.
Retrieve relevant information based on user queries.
Generate accurate answers using an LLM.

* Project Outcome
This project demonstrates the implementation of a Retrieval-Augmented Generation (RAG) pipeline for document-based question answering. It improves response accuracy by combining semantic search with large language models.
